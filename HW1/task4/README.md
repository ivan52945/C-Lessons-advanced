# A3-Извлечение битов

На вход программе подается беззнаковое 32-битное целое число N и натуральное число K (1 ≤ K ≤ 31). Требуется взять K младших битов числа N и вывести полученное таким образом число.

## Пример №1
Данные на входе:	`2 1`  
Данные на выходе:	`0` 

## Пример №2
Данные на входе:	`12 3`  
Данные на выходе:	`4`
